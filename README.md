# FornoGest — Releases

Repositório **público** que hospeda os instaladores e o feed de atualização
automática do **FornoGest**, o ERP para padarias e confeitarias.

> ℹ️ O código-fonte do FornoGest é privado. Este repositório contém **apenas**
> os pacotes de instalação publicados e os metadados usados pelo atualizador
> automático (`latest.yml`).

## ⬇️ Baixar a última versão

Acesse a página de **[Releases](../../releases/latest)** e baixe o instalador
mais recente para Windows (`FornoGest-Setup-x.y.z.exe`).

## 🔄 Atualizações automáticas

Depois de instalado, o FornoGest verifica novas versões automaticamente ao
abrir. Havendo atualização, o app avisa, baixa sob demanda e pede para reiniciar
para instalar — não é preciso baixar manualmente daqui.

## 📦 O que vai em cada release

| Arquivo | Função |
|---|---|
| `FornoGest-Setup-x.y.z.exe` | Instalador para Windows |
| `FornoGest-Setup-x.y.z.exe.blockmap` | Download incremental nas atualizações |
| `latest.yml` | Metadados lidos pelo atualizador (versão, hash, tamanho) |
